# Quality-Evaluation-of-OCR-Recognition-Tasks


# 任务内容

数字化转型深度渗透各行业背景下，手写汉字数字化转换需求日益迫切，教育、政务、企业及个人等多场景均对OCR手写汉字识别准确性提出极高要求，而当前市场上各类OCR手写汉字识别方案受书写风格、人群差异、环境复杂等因素影响，准确性参差不齐，非规范场景下易出现高错误率、漏识错字等问题，严重影响效率甚至造成信息偏差，因此特开展本次评测任务，通过标准化全方位评测体系客观量化各方案性能，精准筛选适配多元场景的高准确性方案。

本次评测任务的核心目标是对不同OCR模型对手写汉字的识别准确性进行量化评估与横向对比，筛选出识别性能更优的方案。通过提供的手写汉字训练集，检验OCR系统对不同书写风格、书写条件、汉字类型的识别能力，为后续OCR技术在手写汉字相关场景的应用选型提供数据支撑。

# 评测数据

本次评测使用的数据集总计300余张PDF，其中训练集为240张PDF，测试集为70张PDF，验证集30张无标签PDF暂不公开。建议所有参赛者对数据进行拆分，将数据1、2、3个字拆分进行训练。

# 评价标准

## 准确度
Accuracy=(TP+TN)/(TP+TN+FP+FN)​
其中，TP表示真阳性，TN表示真阴性，FP表示假阳性，FN表示假阴性。

## F1 Score
F1=2×(Precision×Recall)/(Precision+Recall)
## 精确度

精确度是指模型对于正确预测正类的比例。它可以用来评估二分类问题的性能。精确度的公式为：
Precision=TP/(TP+FP)
## Recall

召回是指模型对于正类的预测比例。它可以用来评估二分类问题的性能。召回的公式为：
Recall=TP/(TP+FN)
## 其他指标

字符准确率CA，精准衡量单字识别的正确性，计算公式如下：
CA=正确识别的字符数/总字符×100%
识别效率：单样本平均识别耗时（单位：毫秒），衡量系统的实时性。

分类错误率：表示模型错误分类的样本比例，是准确性的补数。

对数损失：对数损失度量模型对真实标签的预测概率分布的拟合程度。
